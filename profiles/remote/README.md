# Remote profile samples

This folder holds **documentation and optional sample exports** related to the [remote-profile](../../remote-profile/) admin app.

- **Authoritative device JSON** for bundled offline use lives under [`../local/`](../local/).
- The remote-profile server seeds its database from [`../../remote-profile/server/seed/initial-store.json`](../../remote-profile/server/seed/initial-store.json) on first run; you can export copies here if you want a frozen snapshot for docs or demos.

Optional layout:

- `sample-exported-profiles/` — paste exported JSON from the admin UI for sharing (not required for the demo to run).
